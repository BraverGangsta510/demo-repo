Most Recently Used Algorithm

Let's Explore

 

Description of the algorithm:

MRU works on the principle of “Most Recunt used.when newest page arrived in the main memory and if it not avilable then it replaces Recunt used.MRU page replacement algorithm will behave as Optimal page replacement algorithm.

 

MRU Example:

Consider page reference string 1, 2, 3, 4, 5, 6, 1 , 2, 3, 4, 5, 6 with 4 page frames.Find number of page faults.

  1. Initially all slots are empty, so when 1,2,3,4 came they are allocated to the empty slots —> 4 Page Faults.
  2. when 5 comes, it is not available in memory so it replaces the newest page slot i.e 4 ==> 5 —> 1 Page Faults.
  3. Then 6 comes, it is not available in memory so it replaces the newest page slot i.e 5 ==> 6 .1 Page Fault.
  4. 1 comes, it is available in memory so not increase page fault .0 Page Fault. 2 comes, it is available in memory so not increase page fault .0 Page Fault. 3 comes, it is available in memory so not increase page fault .0 Page Fault.
  5. Then 4 comes, it is not available in memory so it replaces the newest page slot i.e 3 ==> 4 .1 Page Fault. Then 5 comes, it is not available in memory so it replaces the newest page slot i.e 4 ==> 5 .1 Page Fault.
  6. 6 comes, it is available in memory so not increase page fault .0 Page Fault.
  7. Total page fault :- 8 Total page fault Ratio :- 0.66 Total page Hit :- 4 Total page Hit Ratio :- 0.33
